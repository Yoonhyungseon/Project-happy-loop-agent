# VOC 설문 작성 기준서

## 목적
VOC 설문은 금융 서비스 이용 과정에서 고객이 느끼는 불편, 만족, 개선 요구를 수집하기 위한 설문이다.
설문은 고객의 실제 이용 경험을 바탕으로 작성되어야 하며, 특정 답변을 유도하지 않아야 한다.

## 설문 작성 원칙
1. 한 문항에는 하나의 평가 주제만 포함한다.
2. 문항은 짧고 명확하게 작성한다.
3. 금융권 서비스에 맞는 공식적이고 중립적인 문체를 사용한다.
4. 개인정보를 직접 수집하는 문항은 포함하지 않는다.
5. 고객이 실제 경험을 기준으로 답변할 수 있도록 작성한다.
6. 기능 만족도, 사용 편의성, 안정성, 속도, 고객지원, 전반적 만족도를 균형 있게 포함한다.
7. 5점 척도 문항은 긍정·부정 방향이 섞이지 않도록 일관성 있게 작성한다.
8. 주관식 문항은 고객의 구체적 불편사항과 개선 의견을 수집하는 용도로 사용한다.

## 권장 문항 구성
- 5점 척도 문항: 5개 이상
- 주관식 문항: 1개 이상
- 전반적 만족도 문항: 1개
- 개선 의견 문항: 1개

## 5점 척도 옵션
1. 매우 불만족
2. 불만족
3. 보통
4. 만족
5. 매우 만족

## 피해야 할 문항
- 주민등록번호, 계좌번호, 전화번호, 주소 등 개인정보를 직접 요구하는 문항
- 특정 답변을 유도하는 문항
- 한 문항에 여러 평가 요소가 섞인 문항
- 내부 시스템 용어를 고객에게 그대로 노출하는 문항
- 지나치게 긴 문항

## 좋은 문항 예시
- 스타뱅킹 앱에서 원하는 메뉴를 쉽게 찾을 수 있었습니까?
- 스타뱅킹 앱의 이체 절차가 이해하기 쉬웠습니까?
- 스타뱅킹 앱 이용 중 화면 전환 속도에 만족하셨습니까?
- 스타뱅킹 앱의 로그인 및 인증 절차가 편리하다고 느끼셨습니까?
- 스타뱅킹 앱 이용 중 개선이 필요하다고 느낀 점을 자유롭게 작성해 주세요.

## 나쁜 문항 예시
- 스타뱅킹은 매우 편리한데 만족하시나요?
- 로그인, 이체, 조회, 고객센터 기능이 모두 만족스러웠습니까?
- 계좌번호와 연락처를 입력해 주세요.
- 당행의 비대면채널 UX 고도화 방향성에 대해 평가해 주세요.
