# 스타뱅킹 VOC 분류 기준

## 서비스 개요
스타뱅킹은 모바일 환경에서 계좌 조회, 이체, 금융상품 조회, 인증, 알림, 고객지원 등을 제공하는 모바일 뱅킹 서비스이다.

## 주요 VOC 분류

### 로그인/인증
고객이 앱 로그인, 공동인증서, 간편비밀번호, 생체인증, 추가인증 과정에서 느끼는 불편을 포함한다.
예시:
- 로그인이 자주 풀린다.
- 인증 절차가 너무 많다.
- 생체인증이 잘 인식되지 않는다.
- 추가 인증 화면이 이해하기 어렵다.

### 계좌 조회
고객이 계좌 목록, 잔액, 거래내역을 확인하는 과정에서 느끼는 불편을 포함한다.
예시:
- 거래내역을 찾기 어렵다.
- 계좌 목록이 복잡하다.
- 잔액 확인 화면이 직관적이지 않다.

### 이체
고객이 이체, 예약이체, 자주 쓰는 계좌, 이체한도 확인 과정에서 느끼는 불편을 포함한다.
예시:
- 이체 단계가 많다.
- 수취인 확인 화면이 불편하다.
- 자주 쓰는 계좌 등록이 어렵다.
- 이체 완료 후 확인 정보가 부족하다.

### 메뉴 탐색
고객이 원하는 메뉴나 기능을 찾는 과정에서 느끼는 불편을 포함한다.
예시:
- 메뉴가 너무 많다.
- 검색을 해도 원하는 기능이 잘 나오지 않는다.
- 자주 쓰는 기능으로 바로 가기 어렵다.

### 앱 속도
앱 실행, 화면 전환, 조회, 이체 처리 속도와 관련된 불편을 포함한다.
예시:
- 앱 실행이 느리다.
- 화면 전환이 늦다.
- 이체 처리 중 로딩이 길다.

### 안정성
앱 오류, 강제 종료, 멈춤, 접속 실패, 장애와 관련된 불편을 포함한다.
예시:
- 앱이 갑자기 종료된다.
- 화면이 멈춘다.
- 접속 오류가 반복된다.

### 화면 UX
화면 구성, 버튼 위치, 글자 크기, 안내 문구, 입력 편의성과 관련된 불편을 포함한다.
예시:
- 버튼 위치가 헷갈린다.
- 안내 문구가 어렵다.
- 글씨가 작다.
- 입력해야 할 항목이 많다.

### 고객지원
고객센터 연결, FAQ, 챗봇, 상담 안내와 관련된 불편을 포함한다.
예시:
- 고객센터 연결 방법을 찾기 어렵다.
- FAQ 답변이 충분하지 않다.
- 오류 발생 시 안내가 부족하다.

### 알림/푸시
입출금 알림, 보안 알림, 마케팅 알림, 푸시 수신과 관련된 불편을 포함한다.
예시:
- 알림이 너무 자주 온다.
- 필요한 알림이 오지 않는다.
- 알림 설정 위치를 찾기 어렵다.

## 설문 생성 시 우선 포함할 영역
스타뱅킹 VOC 설문은 기본적으로 아래 영역을 우선 포함한다.
1. 로그인/인증
2. 계좌 조회
3. 이체
4. 메뉴 탐색
5. 앱 속도
6. 안정성
7. 화면 UX
8. 고객지원
9. 전반적 만족도
10. 개선 의견

## 카테고리 코드 예시
LOGIN_AUTH: 로그인/인증
ACCOUNT_VIEW: 계좌 조회
TRANSFER: 이체
MENU_NAVIGATION: 메뉴 탐색
APP_SPEED: 앱 속도
STABILITY: 안정성
UX_UI: 화면 UX
CUSTOMER_SUPPORT: 고객지원
NOTIFICATION: 알림/푸시
OVERALL: 전반적 만족도
